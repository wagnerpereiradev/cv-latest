# Wagner Pereira | CV

Site de currículo e portfólio em formato web, com design minimalista inspirado na Apple.

## Sobre o projeto

Página única (single-page) que reúne:

- **Perfil** — resumo profissional, hard e soft skills
- **Stack** — tecnologias e ferramentas (IA, frontend, backend, cloud, produto)
- **Projetos com IA** — agentes e produtos (Sifat Chatbot, Agente do Clima, Conta Black, LajeAI)
- **Experiência** — trajetória profissional
- **Formação** — cursos e habilidades desenvolvidas
- **Eventos** — hackathons, imersões e programas de capacitação

Desenvolvido com **Next.js**, **React** e **Tailwind CSS**, sem templates prontos. Código limpo, responsivo e acessível.

## Como rodar

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## Deploy

O projeto pode ser implantado na [Vercel](https://vercel.com) ou em qualquer plataforma que suporte Next.js.
